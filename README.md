Aterrizaje HZ 🚀

Landing HZ es una página de destino moderna y responsiva diseñada para presentar servicios digitales como desarrollo de software a medida, soporte técnico, mantenimiento de sistemas y ensamblaje de hardware . El proyecto se centra en una interfaz de usuario limpia, una presentación clara de los servicios y un diseño profesional, ideal para agencias o desarrolladores freelance.


🛠 Tecnologías utilizadas

-Vite

-TailwindCSS

-JavaScript / TypeScript

-HTML5

-CSS3

Estas herramientas permiten un desarrollo rápido, compilaciones optimizadas y un flujo de trabajo de interfaz moderno.
